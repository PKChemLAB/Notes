

# 紫外-可见分光光度法（Ultraviolet - Visible Spectrophotometry, UV-Vis）
## 第一节 紫外-可见吸收光谱
### 一、定义
紫外-可见吸收光谱，是使紫外-可见区（200 nm ~ 800 nm）不同波长的光依次通过样品溶液，记录每一波长下样品的吸光度，以波长为横坐标、吸光度为纵坐标绘制得到的吸收曲线（吸收光谱）。

### 二、光谱机理
1. 紫外-可见吸收光谱属于**分子吸收光谱**，由分子中**价电子能级跃迁**产生（基于分子轨道理论）。

2. 紫外-可见区有机吸收光谱**主要由$\boldsymbol{\pi→\pi^*}$和$\boldsymbol{n→\pi^*}$跃迁**产生：

    | 跃迁类型 | 对应基团示例 | 最大吸收波长λmax | 摩尔吸收系数ε |
    |----------|--------------|------------------|---------------|
    | $\sigma→\sigma^*$ | C—C单键 | ＜150 nm | ~10⁴ |
    | $\sigma→\pi^*$ / $\pi→\sigma^*$ | - | ＜200 nm | - |
    | $\pi→\pi^*$ | C=C、C=O双键 | 180-400 nm（可延伸至可见区） | ＞10⁴（强吸收） |
    | $n→\sigma^*$ | C—X（X=O,S,N,P,卤素） | ＜200 nm | 10¹~10² |
    | $n→\pi^*$ | C=X（X=O,S,N等） | ~300 nm | 10¹~10²（弱吸收） |

### 三、光谱特点
1. 分子吸收光谱为**带状光谱**（理论上为线光谱），吸收峰宽度可达几十nm。
2. 带状光谱成因：分子运动的复杂性
    - 分子的运动方式包含价电子运动、原子核间相对振动、分子旋转，对应三类能级：价电子能级、分子振动能级、分子转动能级。
    - 能级差规律：
     $\Delta E_{\text{价电子}} : \Delta E_{\text{振动}} : \Delta E_{\text{转动}} = 1 : 1/10 : 1/100$

    - 分子发生价电子能级跃迁时，不可避免伴随振动和转动能级的跃迁，相邻谱线间隔极小（~0.25 nm），大量谱线相连形成宽吸收峰，即带状光谱。

### 四、常用术语
- **最大吸收波长（λmax）**：吸收峰最高点对应的波长。
- **生色团**：分子中能吸收紫外-可见光的结构单元，多为含不饱和键的基团，如C=C、C=O等。
- **助色团**：本身不能吸收紫外-可见光，但能使生色团的λmax红移、摩尔吸收系数ε增大的基团，如-OH、-COOH等。
- **红移（长移）**：吸收峰的λmax向长波长方向移动的现象。
- **蓝移（短移）**：吸收峰的λmax向短波长方向移动的现象。

## 第二节 紫外-可见分光光度法定量原理
### 一、朗伯-比尔定律
1. 定律发展历程
    - 1729年布格、1760年朗伯：阐明光吸收程度与吸收介质厚度的关系，$A = k'l$
    - 1852年比尔：阐明光吸收程度与吸光物质浓度的关系，$A = k''c$
    - 二者结合得到**布格-朗伯-比尔定律**，简称朗伯-比尔定律，是光吸收的基本定律。

2. 
    $$A = kcl$$

    辅助定义式：

    $$A = \lg\frac{I_0}{I} = -\lg T$$

    - $A$：吸光度
  
    - $I_0$：入射光强度
  
    - $I$：透射光强度

    - $T$：透过率（透射比），$T=I/I_0$
  
    - $k$：吸光系数
  
    - $c$：吸光物质的浓度
  
    - $l$：光程（吸收池厚度）
  
3. 适用前提：一定波长的**平行单色光**照射稀溶液体系。

### 二、吸光系数
1. **定义**：吸光物质在单位浓度、单位光程厚度时的吸光度，是物质吸光能力的核心量度。
2. **主要类型**
   
| 类型 | 符号 | 定义 | 单位 |
| :--- | :--- | :--- | :--- |
| 摩尔吸光系数 | $\varepsilon$ | 吸光物质浓度为1 mol/L、光程为1 cm时的吸光度 | $\text{L·mol}^{-1}·\text{cm}^{-1}$ |
| 吸光系数 | $a$ | 吸光物质浓度为1 g/L、光程为1 cm时的吸光度 | $\text{L·g}^{-1}·\text{cm}^{-1}$ |

3. **核心公式**
    - 定义式：$\varepsilon = \frac{A}{cl}$
    - 测量式：$A = \varepsilon cl$
    - 意义式：$\varepsilon = 0.87 × 10^{20} · B · E$，其中$E$为吸光粒子有效面积，$B$为跃迁概率（由待测物结构决定）
4. **物理意义**
    1. 是吸光物质吸光能力（检测灵敏度）的量度：
        - $\varepsilon$ ~10²：弱吸收
        - $\varepsilon$ ~10⁴：中吸收
        - $\varepsilon$ ~10⁵：强吸收
    2. 是物质的特性常数：在固定波长、温度、溶剂条件下为定值，可作为定性指标之一；同一物质对不同波长光的$\varepsilon$不同，不同物质对同一波长光的$\varepsilon$通常不同。
    3. 吸光度具有加和性：多组分体系中，总吸光度等于各组分在该波长下的吸光度之和。

### 三、偏离朗伯-比尔定律的因素
1. **现象**：理想状态下吸光度A与浓度c呈线性关系，标准曲线为直线；实际检测中标准曲线发生弯曲，A与c偏离线性关系。
2. **核心原因**
    - **化学因素（样品因素）**
      朗伯-比尔定律仅适用于**稀溶液（＜10⁻² mol/L）**；当溶液浓度过高时，分子间相互作用显著增强，导致吸光性质改变，曲线发生弯曲。
    - **光学因素**
      定律理论要求入射光为平行单色光；但实际光源为复合光，经单色器分光后得到的是具有一定波长范围的复合光（非严格单色光），从而导致对定律的偏离。

## 第三节 紫外-可见分光光度计
### 一、仪器基本组成
紫外-可见分光光度计由五大核心部分构成，光路顺序为：
**光源 → 单色器 → 吸收池 → 检测器 → 信号显示系统**

#### 1. 光源
提供稳定、高强度的连续复合光，核心类型及参数：
| 光源类型 | 发光原理 | 适用波长范围 | 核心适用波段 |
|----------|----------|--------------|--------------|
| 钨灯/卤钨灯 | 热辐射光源 | 320-2500 nm | 可见光区 |
| 氢灯/氘灯 | 气体放电灯 | 180-375 nm | 紫外光区 |
| 氙灯 | 气体放电灯 | 250-700 nm | 紫外-可见双区 |

#### 2. 单色器
核心作用：将光源发出的复合光色散为所需波长的单色光（一定谱带宽度）。
核心色散元件：
- **滤光片**：分为吸收滤光片（有色玻璃，半宽度20-50 nm）、干涉滤光片（半宽度~10 nm），按功能分为带通型和截止型。
- **棱镜**：基于光的折射原理，不同波长光折射率不同；特点为非线性色散、非匀排光谱，目前应用较少。
- **光栅**：基于光的干涉和衍射原理，为高度抛光平面刻有大量平行等距槽（300-2000条/mm）；特点为色散能力强、分辨率高、匀排光谱，是目前主流的色散元件。

#### 3. 吸收池（比色池/比色皿）
- 用途：盛放样品溶液的专用容器。
- 材质与适用范围：
  - 玻璃材质：仅用于可见光区（玻璃会吸收紫外光）。
  - 石英材质：可用于紫外-可见全波段。
- 常用规格：光程1 cm、2 cm、5 cm，其中1 cm为最通用规格。
- 核心要求：需具备匹配性（配对），即同一组吸收池需保持相同的厚度和透光性能。

#### 4. 检测器
光电转换装置，基于光电效应将光信号转换为可测量的电信号，核心类型：
| 检测器类型 | 核心特点 | 适用场景 |
|------------|----------|----------|
| 硒光电池/硅光电池 | 硒光电池易疲劳、灵敏度低，已少用；硅光电池为当前主流替代款 | 300-800 nm可见光区，简易型仪器 |
| 光电二极管（光电管） | 灵敏度高、不易疲劳；分紫敏（210-625 nm）、红敏（625-1000 nm） | 紫外-可见区常规检测 |
| 光电倍增管（PMT） | 基于二次电子倍增效应，灵敏度极高（增益10⁵~10⁸），适用于弱光检测 | 高精度、痕量检测 |
| 阵列检测器（PDA/CCD/CMOS） | 光电二极管阵列（PDA）、电荷耦合器件（CCD），可多波长同时检测，无需扫描即可快速获得全波段吸收光谱 | 在线检测、化学反应动力学研究、多组分同时分析 |

#### 5. 信号处理显示系统
功能：对检测器输出的电信号进行处理、计算、显示与记录。
常见形式：指针表头、数字显示屏、记录仪、电脑数据处理系统、打印机等。

### 二、仪器类型
#### 1. 单光束分光光度计
- 结构：光源→单色器→吸收池→检测器→信号显示系统，单一路径光束。
- 特点：光路简单、成本低；无法校正光源强度波动，不具备自动波长扫描功能，不能自动测量吸收光谱。

#### 2. 双光束分光光度计
- 结构：单色器输出的单色光经斩光器/旋转装置，交替通过空白参比池和样品池，最终到达检测器。
- 特点：可自动扣除参比背景，自动消除光源强度变化带来的误差，支持自动波长扫描与吸收光谱测定；结构较复杂，价格相对较高。

#### 3. 双波长分光光度计
- 结构：两个不同波长的单色光交替通过同一个样品池，测定两个波长下的吸光度差值ΔA。
- 特点：可有效消除背景吸收、共存组分的干扰；结构更复杂，价格更高。

#### 4. 多道型分光光度计
- 结构：采用阵列检测器，可同时检测广域多波长的光吸收信号。
- 特点：无需波长扫描，可瞬间获得全波段吸收光谱，检测速度快；适用于在线检测、化学反应动力学研究、多组分同时分析。
