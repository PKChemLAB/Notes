# Database of Physical Chemistry

| 数据库 | 全称 | 简介 |
|---|---|---|
| [CCCBDB](https://cccbdb.nist.gov/) | Computational Chemistry Comparison and Benchmark DataBase | NIST 旗下的计算化学基准数据库，擅长气相小分子的结构数据（转动能级、键长、振动频率等），可与实验和高精度计算交叉验证 |
| [NIST Webbook](https://webbook.nist.gov/) | National Institute of Standard and Technology Chemistry Webbook | 权威且免费的热力学与光谱数据库，优势在于数据量大、覆盖面广，尤其红外和质谱图收录丰富 |
| [CRC Handbook](https://hbcp.chemnetbase.com/) | CRC Handbook of Chemistry and Physics | 百年经典手册（1913 首版），传统物化数据最全最权威，支持离线和在线检索，但需要机构授权 |
| [CCDC](https://www.ccdc.cam.ac.uk/) | Cambridge Crystallographic Data Centre | 晶体学领域的事实标准，收录全球已发表的小分子和金属有机晶体结构，支持子结构检索与统计分析 |
| [上海有机化学所](http://www.organchem.csdb.cn/) | 上海有机化学研究所数据库 | 国产免费化学数据库，谱学数据（核磁、红外、质谱）齐全，有机化合物覆盖好，外网注册即可使用 |
